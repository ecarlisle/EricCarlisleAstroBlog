# Project Overview

AstroBlog is Eric Carlisle's personal site for technical writing, portfolio work, and professional
presence.

The site supports:

- Blog articles for software developers
- Portfolio and case study content
- Search-friendly professional visibility
- Lightweight contact and lead generation
- High-performance static delivery

The experience should feel:

- Fast
- Thoughtful
- Human
- Technical
- Accessible
- Professional without becoming sterile

The current codebase is close to feature-complete. Some existing content is placeholder content used
to test layout, styling, tags, pagination, search, and metadata. It should be replaced deliberately
without removing the test coverage those posts currently provide.
